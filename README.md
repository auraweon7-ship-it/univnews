# 한국 사립대학교 뉴스 대시보드

> 5대 유형 분류 + 관점 + 지역별 사립대 이슈 모니터링 대시보드

![Dashboard Preview](https://img.shields.io/badge/뉴스-57건-6c5ce7?style=for-the-badge) ![Sources](https://img.shields.io/badge/언론사-23개-0891b2?style=for-the-badge) ![Categories](https://img.shields.io/badge/세부항목-20개-10b981?style=for-the-badge)

## 개요

한국 사립대학교 관련 주요 뉴스와 이슈를 **5대 유형, 20개 세부항목**으로 분류하여 제공하는 단일 HTML 대시보드입니다. 네이버뉴스 기반으로 수집된 57건의 뉴스를 관점(진보/중도/보수), 지역(수도권/충청/호남/영남/강원·제주), 기간별로 필터링할 수 있습니다.

## 5대 유형 분류 체계

| 대유형 | 세부항목 | 뉴스 수 |
|--------|---------|---------|
| 💰 **재정 건전성** | 등록금·등록금의존율, 법인전입금, 적립금·이월금, 부채·재정위기 | 15건 |
| 📚 **교육 투자** | 학생1인당 교육비, 교육비환원율, 장학금, 교원확보율 | 9건 |
| 🎓 **학생 경쟁력** | 충원율, 중도탈락률, 취업률, 외국인 유학생 | 10건 |
| 🏛️ **법인 책무성** | 이사회·수익용기본재산, 감사·정보공시, 교비횡령·비리, 사립학교법 | 9건 |
| 🌱 **지속가능성** | 학령인구 대응, 구조개혁·구조개선법, 지역연계·생존전략, AI·디지털 전환 | 14건 |

## 주요 기능

### 필터링
- **관점 필터**: 전체 / 진보 / 중도 / 보수 (언론사 기반 자동 분류)
- **지역 필터**: 전체 / 수도권 / 충청권 / 호남권 / 영남권 / 강원·제주 / 전국·정책
- **기간 설정**: 달력 선택 + 프리셋 버튼 (1개월 / 3개월 / 6개월 / 전체)
- **키워드 검색**: 제목·본문·언론사명 실시간 검색
- **정렬**: 최신순 / 오래된순

### 2단계 카테고리
- **1단계**: 5대 유형 (재정 건전성, 교육 투자, 학생 경쟁력, 법인 책무성, 지속가능성)
- **2단계**: 20개 세부항목 (등록금, 법인전입금, 적립금, 충원율, AI·디지털 전환 등)

### 뉴스 카드 정보
- 뉴스 제목 (이모지 포함, 원문 링크)
- 150자 이내 요약
- 출처 언론사, 키워드 태그
- 관점 배지 (진보 🔵 / 중도 🟣 / 보수 🔴)
- 지역 배지
- 📷 표지이미지 URL (복사 버튼)
- 🔗 출처 URL (복사 버튼)

### 통계 바
- 표시 뉴스 수, 진보/중도/보수 비율, 최다 지역, 언론사 수, 최신 기사일

## 실행 방법

### 방법 1: 파일 직접 열기
```
index.html을 브라우저에서 직접 열기 (더블클릭)
```

### 방법 2: Node.js 서버
```bash
node server.js
# http://localhost:3200 에서 접속
```

## 파일 구조

```
├── index.html    # 대시보드 본체 (단일 HTML, 외부 의존성 없음)
├── server.js     # 간단 HTTP 서버 (Node.js)
└── README.md
```
